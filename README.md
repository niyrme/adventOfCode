### expand script
- Requires template in `YEAR/00`

| Flags | Behaviour |
|----------------------------|-------------------------------|
| none                       | add next day to current year  |
| `-d DAY` or `--day DAY`    | add day `DAY` to current year |
| `-y YEAR` or `--year YEAR` | add next day to year `YEAR`   |
| `-d DAY -y YEAR`           | add day `DAY` to year `YEAR`  |
